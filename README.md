### 🎮 Sobre o PXBR

Há mais de **5 anos** construindo o Pixelmon Brasil, de um servidor de Minecraft a uma marca multiplataforma com:

- 🏆 **+1 milhão** de downloads na Playstore
- 🌐 Rede multi-servidor rodando SpongeForge 1.12.2 + Pixelmon Reforged
- ⚙️ Ecossistema próprio de plugins
- 💬 Comunidade ativa no Discord

### 🛠️ O que eu faço

- Desenvolvimento de **plugins Java** para SpongeForge / Pixelmon
- Arquitetura e administração de infraestrutura multi-servidor
- Tuning de JVM/GC, otimização de performance
- Automação de backups, CI/CD e ferramentas internas
- Estratégia de produto, monetização e marketing para games

### 🧰 Stack principal

```
Linguagens    →  Java, Kotlin, TypeScript, Bash
Minecraft     →  SpongeForge, Forge
Backend       →  HikariCP, Jankson, JDA, Node.js
Dados         →  MariaDB, MongoDB, MySQL, PostgreSQL, Redis
Infra         →  Linux, Docker, Cloudflare
```
